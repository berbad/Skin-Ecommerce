## Skin E-Commerce Website

A modern skincare e-commerce platform focused on simplicity and user experience. This project allows users to browse, search, and purchase skincare products.

## Features
- Product listing with detailed descriptions and images  
- Search and filtering by skin type, category, or brand  
- User authentication (signup/login)  
- Shopping cart and checkout flow  
- Order tracking and user dashboard  
- Fully responsive design (mobile + desktop)
- Payment integration (Stripe)
- Admin dashboard for inventory management 

## Tech Stack
- Frontend: (React, Next.js)  
- Backend: (Node.js, Express)  
- Database: (MongoDB)  
- Authentication: (JWT, OAuth)
